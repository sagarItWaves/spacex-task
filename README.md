# SpaceX React Task

### Prerequsite

- node v16.14.2
- ReactJS 18.2.0

### Run the project by following below commands

- `git clone  https://github.com/sagarItWaves/spacex-task.git`
- `cd spacex-task`
- `npm install`
- `npm run start`

### Eslint Configuration

- ESLink configuration is available inside .eslintrc file in project root directory.
